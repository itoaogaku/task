# iPhone ホーム画面ウィジェット（Scriptable）

未完了タスクを優先度順にホーム画面へ表示するウィジェットです。アプリ本体と
同じ GAS バックエンド（スプレッドシート）を読むので、常に同じ内容になります。

## セットアップ

1. App Store で無料アプリ **Scriptable** をインストール
2. Scriptable を開き、右上の **＋** で新規スクリプトを作成
3. [`scriptable-widget.js`](./scriptable-widget.js) の中身をすべて貼り付けて保存
   （スクリプト名は「タスク」などわかりやすい名前に）
4. ホーム画面を長押し → **＋** → **Scriptable** を選び、小・中・大のいずれかを追加
5. 追加したウィジェットを長押し → **ウィジェットを編集** →
   **Script** に貼り付けたスクリプトを選択

## 表示

- 未完了タスクを優先度順（アプリと同じ並び）に表示
- 件数はサイズで自動調整：小 3 / 中 6 / 大 12 件
- 各行に優先度バッジ（色つき）、中・大サイズでは確認先も表示
- ウィジェットをタップするとアプリ（Vercel）が開く
- 約 30 分ごとに自動更新（iOS の判断で前後します）

## カスタマイズ

`scriptable-widget.js` 冒頭の設定で変更できます。

| 変数 | 内容 |
| --- | --- |
| `API_URL` / `TOKEN` | GAS のURL・トークン（`config.js` と同じ値） |
| `APP_URL` | タップで開くアプリのURL |
| `ACCENT` | アクセントカラー |
| `REFRESH_MINUTES` | 更新間隔（分） |

> メモ: `API_URL` と `TOKEN` は公開中の `config.js` と同じ値です。
> ウィジェット化で新たに公開範囲が広がるものではありません。
